# Apna Invoice - Project Management & CRM System

## Project Overview

Apna Invoice is a full-stack Project Management and CRM platform built using React, TypeScript, Node.js, Express.js, and PostgreSQL.

The system helps organizations manage projects, team members, tasks, client relationships, analytics, calendars, and payments from a centralized dashboard.

# Installation Guide

## Frontend

```bash
npm install
npm run dev
```

## Backend

```bash
npm install
npm run dev
```

## Database

Configure PostgreSQL and update:

```env
DATABASE_URL=
JWT_SECRET=
PORT=5001
```
